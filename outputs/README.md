outputs

This folder stores pipeline-generated outputs.

results
tables and csv files such as summaries, metrics, predictions, coefficients, and feature importance

figures
ECG examples, ROC curves, confusion matrices, and preprocessing visualizations

splits
train, val, and test split files

Important files include:
- results/data_summary.csv
- results/experiment_summary_val.csv
- results/experiment_summary_test.csv
- results/lr_coefficients_balanced.csv
- splits/train.csv
- splits/val.csv
- splits/test.csv

All files in this folder are generated by the script pipeline.
