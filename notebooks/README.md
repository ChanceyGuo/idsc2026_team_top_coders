notebooks

This folder stores the main Kaggle notebook used to run the full pipeline.

Recommended notebook:
IDSC2026_TEAM_TOP_CODERS_MAIN.ipynb

Purpose

The notebook serves as the main execution entry point in Kaggle.
It clones the GitHub repository, sets dataset paths, and runs the pipeline scripts in order.

Recommended usage

Run the notebook in Kaggle.
Use the official dataset input path.
Execute the cells in sequence from environment setup to model evaluation.

The notebook is intended to support reproducibility and make the full workflow easier to inspect.
